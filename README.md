# VizeKit - PHP Version

## FTP Upload Dosyaları

Hostware hosting hesabınıza yüklemek için gerekli dosyalar:

### public_html klasörüne yükleyin:
- `index.php`
- `.htaccess` 
- `pages/` klasörü (tüm alt dosyalarla)

### Dosya Yapısı:
```
public_html/
├── index.php
├── .htaccess
└── pages/
    ├── home.php
    ├── visa-checker.php
    ├── countries.php
    ├── services.php
    └── contact.php
```
